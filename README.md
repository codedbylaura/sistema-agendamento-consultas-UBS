# sistema-agendamento-consultas-UBS
Sistema de Agendamento de Consultas para uma Unidade Básica de Saúde (UBS). O projeto permite cadastrar pacientes, profissionais de saúde e consultas, facilitando a organização dos atendimentos. Além disso, gera relatórios simples para auxiliar no acompanhamento e na gestão dos serviços, contribuindo para um atendimento mais rápido e eficiente.
